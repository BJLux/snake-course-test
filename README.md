# Python Snake Game

A classic Snake game built with Python and Pygame.

## Features
- Arrow key controls
- Score tracking
- Growing snake
- Game over and restart functionality

## Setup Instructions

1.  **Install Python:** Ensure you have Python 3.6 or later installed.
2.  **Install Dependencies:** Run the following command to install Pygame:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the Game:** Execute the game script:
    ```bash
    python snake.py
    ```

## Controls
- **Up Arrow:** Move Up
- **Down Arrow:** Move Down
- **Left Arrow:** Move Left
- **Right Arrow:** Move Right
- **Q:** Quit (on Game Over screen)
- **C:** Play Again (on Game Over screen)
